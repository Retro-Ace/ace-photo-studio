# ACE Photo Studio v1.1.1

Maintenance release focused on stability, responsiveness, and release hygiene.

## Included in this release

- Large HDR batch workflows now stay more responsive by chunking library insertion and deferring unnecessary full thumbnail hydration.
- Release-facing docs and version metadata are aligned to v1.1.1.
- Existing merge isolation, preview/export parity, compare, presets, reset, and tone curve behavior remain unchanged.

## Notes

- This release does not introduce a new editor feature set.
- Normal export, HDR merge, and library workflows should behave the same as before, with better responsiveness on large batches.
